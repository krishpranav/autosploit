# autosploit
A simple ruby tool to automate metasploit modules

[![forthebadge](https://forthebadge.com/images/badges/made-with-ruby.svg)](https://forthebadge.com)

# Installation
```
git clone https://github.com/krishpranav/autosploit
cd autosploit
bash autosploit.sh
```
